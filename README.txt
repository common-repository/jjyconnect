== Description ==

### 集简云-无代码软件集成平台

集简云平台可快速将您的软件与数百款软件集成连接，上架到集简云平台无需再与每个软件进行独立的集成开发。

集简云平台可连接各类电商、销售、客服、办公、社交等应用服务，包括**抖音**，**企业微信**，**钉钉**，**用友YonSuite系统**，**金蝶云星空**，**企业API接口**等大型企业系统等实现数据互通。

###集简云功能特点

* 大幅节约开发时间
* 自动扩展与新软件的连接
* 获取更多销售线索
* 让您的客户更满意
* 满足不同的使用场景
* 无需任何费用

###集简云自动化数据流程使用场景

* 对市场团队
 销售线索信息自动同步，个性化推广信息，自动线索评分，推广数据自动收集，线索信息扩展，自动化营销流程

* 对销售团队
CRM系统信息同步，潜在成交机会预警，关键信息自动预警，自动化审批流程，市场与销售信息同步，客服与CRM系统同步信息

* 对服务团队
自动响应时效设置，自动服务优先级，同步销售/市场/服务数据，用户流失预警，自动记录服务信息

* 对数据分析团队
自动汇集多个系统间的数据，关键数据自动收集，数据流程监控，自定义数据流程规则

###上架流程，我们全程支持

* 注册并提交应用信息
提交您所拥有的应用名称和描述，进行[集简云注册](https://apps.jijyun.cn/register)

* 创建接口信息
将您的接口同步到集简云，创建应用授权，触发动作或者执行动作。

* 发布内测
提交您的应用，我们的工作人员将进行测试，您也可以邀请制定企业用户测试。

* 全网发布
在经过我们工作人员的测试后即可发布全网，所有集简云的用户均可使用。

* 市场推广
我们的工作人员将帮助您策划营销推广和流程示例，让更多集简云的用户了解和使用您的产品。

== Frequently Asked Questions ==
常见问题1：目前有很多软件厂商想上架到集简云，但是排期时间较久，是否可以自己完成上架？
 答：可以通过集简云开发者平将自己企业的软件上架到集简云，省时省力，且无需任何费用。

常见问题2： 如何查看我的流程是否执行成功？失败了如何发现？
答：可以通过集简云平台左侧菜单“流程日志”查看流程执行状态。提供日期和流程状态筛选。每一条流程日志点击后都会进入日志的详细界面，可以查看流程的失败原因。

常见问题3：集简云如何收费以及计费标准是什么？
答：集简云是按执行次数（即接口调用的次数）计费的，不同版本包含不同数量的执行次数、以及最大流程和应用数量，和免费接口开发的个数，
目前有免费版本、基础版本、团队协作版本、企业协作版本、企业高级版本和定制版本供客户选择，具体版本信息和资源情况可以在集简云官网[“产品价格”](https://www.jijyun.cn/pricing.html)中查看。
